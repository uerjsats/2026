Atividade UERJ Sats

Eletrônica:
Reproduza a placa de computador de bordo no Kicad, os componentes necessários são:
  - Gy-Neo6mv2 (GPS)
  - DHT22
  - Heltech ESP 32 Lora
  - BMP280
  - Gy-521
  - Eeprom I2c

OBS: Baixe as bibliotecas do Kicad no drive da Sats.

Como entregar?

Colocar a atividade na pasta do drive do seu subsistema (criar uma pasta com seu nome e atividade).

Computador de bordo:
Faça um código utilizando o Esp 32 para receber os dados de um Gy-Neo6mv2 (GPS) e um DHT22 e printe os dados no Esp 32 para ele confirmar que recebeu os dados. 

Como entregar?

Upe o código para a o repositório de capacitações no GitHub, link: 2026/Capacitações 26.1 at main · uerjsats/2026
